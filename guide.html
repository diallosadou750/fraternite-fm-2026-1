🚀 Mise en ligne sur Netlify

Le site est prêt à être déployé. Suivez ces 3 étapes dans l'ordre.

Étape 1 — Créer un compte GitHub

1
Créez un compte sur github.com
Choisissez un nom d'utilisateur simple. Ex: fraternitefm
2
Créez un nouveau dépôt (repository)
Cliquez "New" → Nom: fraternite-fm → Visibilité: Public → "Create repository"
3
Uploadez tous les fichiers
Sur la page du dépôt → "uploading an existing file" → glissez TOUT le contenu du dossier fraternite-fm/ → "Commit changes"

Étape 2 — Déployer sur Netlify

1
Connectez-vous sur netlify.com avec GitHub
Cliquez "Sign up with GitHub" → autorisez l'accès
2
Importez votre dépôt
"Add new site" → "Import an existing project" → GitHub → choisissez fraternite-fm
3
Déployez
Laissez tout par défaut → "Deploy site" → votre site est en ligne en 2 minutes ✅
4
Notez votre URL
Netlify vous donne une URL du type https://random-name.netlify.app. Vous pouvez la personnaliser dans Site Settings → Domain.

Étape 3 — Activer le CMS Decap

1
Créez une OAuth App GitHub
GitHub → Settings → Developer settings → OAuth Apps → "New OAuth App"
• Application name: Fraternité FM CMS
• Homepage URL: votre URL Netlify
• Callback URL: https://sveltia-cms-auth.netlify.app/callback
2
Mettez à jour admin/config.yml
Dans GitHub, ouvrez admin/config.yml → modifiez la ligne repo: VOTRE_USERNAME/fraternite-fm → mettez votre vrai pseudo GitHub
3
Accédez à l'éditeur
Allez sur votre-site.netlify.app/admin/ → connectez-vous avec GitHub ✅

🔔 Notifications Push (OneSignal)

Les notifications push permettent d'alerter vos abonnés quand une émission commence ou quand un article important est publié. C'est gratuit jusqu'à 10.000 abonnés.

💡 Pourquoi c'est crucial pour votre objectif 20.000 usersLes auditeurs qui activent les notifications reviennent 5× plus souvent que ceux qui ne les activent pas. C'est votre levier de fidélisation le plus puissant.
1
Créez un compte sur onesignal.com
Gratuit — choisissez "Web Push" → entrez l'URL de votre site Netlify
2
Copiez votre App ID
Dans OneSignal → Settings → Keys & IDs → copiez le OneSignal App ID
3
Collez l'ID dans index.html
Ouvrez index.html sur GitHub → cherchez YOUR_ONESIGNAL_APP_ID → remplacez par votre vrai ID → sauvegardez → Netlify redéploie automatiquement
4
Envoyez votre première notification
OneSignal Dashboard → "New Push" → rédigez votre message → "Send to All" → vos abonnés sont alertés en temps réel 🔔
✅ Le site est déjà configuréLe fichier OneSignalSDKWorker.js est déjà présent à la racine. Le bandeau "Activer les notifications" s'affiche automatiquement après 8 secondes. Il ne vous manque que l'App ID.

📰 Publier un article

Procédure quotidienne recommandée : 2 articles par jour minimum.

1
Ouvrez le CMS → "📰 Articles & Actualités" → "New Articles"
2
Remplissez le formulaire
Titre percutant · Catégorie · Date · Icône · Résumé (2-3 phrases)
3
Cliquez "Publish" en haut à droite
L'article est créé sur GitHub → Netlify redéploie → visible sur le site en ~30 secondes
4
Partagez immédiatement
Copiez le lien de la section Actualités → publiez sur Facebook, WhatsApp, Instagram
⚠️ Ne pas oublierAprès chaque publication, mettez à jour manuellement le fichier _data/articles/index.json sur GitHub en ajoutant le nom du nouveau fichier JSON créé par le CMS. Sans ça, l'article n'apparaîtra pas sur le site.

🎬 Ajouter un clip vidéo

1
Uploadez votre clip sur YouTube
Connectez-vous sur youtube.com → "Créer" → "Mettre une vidéo en ligne" → choisissez votre fichier
2
Copiez l'ID YouTube
Dans l'URL du clip : youtube.com/watch?v=dQw4w9WgXcQ → l'ID est la partie après v=
3
CMS → "🎬 Clips & Replays" → "New Clip"
Titre · Catégorie · Durée · Icône · ID YouTube (copiez-le ici) · Description · Ordre
4
Publiez → le clip apparaît sur le site avec lecteur intégré
Le clic sur un clip ouvre un modal de lecture directement sur la page

📅 Modifier les programmes

CMS → "📅 Grille des Programmes" → cliquez sur une émission → modifiez → "Save".

💡 Ordre d'affichageLe champ "Ordre" détermine la position de l'émission dans la grille. 1 = premier, 9 = dernier. Gardez-le cohérent avec les horaires.

👥 Gérer l'équipe

CMS → "👥 Équipe" → cliquez sur un membre ou "New Membre".

Pour ajouter une photo : dans le CMS, cliquez sur le champ "Photo" → uploadez une image JPG/PNG → sauvegardez. L'image remplace automatiquement les initiales.

🔍 SEO & Référencement Google

Votre site est déjà optimisé avec Schema.org, sitemap.xml et balises meta. Voici les actions manuelles pour accélérer l'indexation.

1
Soumettez votre sitemap à Google
Allez sur search.google.com/search-console → ajoutez votre site → "Sitemap" → entrez /sitemap.xml → "Soumettre"
2
Créez votre fiche Google Business
Allez sur business.google.com → "Ajouter une entreprise" → catégorie "Station de radio" → Mamou, Guinée → vérifiez
3
Mettez à jour l'URL dans sitemap.xml
Ouvrez sitemap.xml sur GitHub → remplacez VOTRE-SITE.netlify.app par votre vraie URL → sauvegardez
4
Faites de même dans robots.txt
Ouvrez robots.txt → remplacez l'URL du Sitemap → sauvegardez
